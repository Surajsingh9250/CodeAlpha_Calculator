# CodeAlpha_Calculator

Modern Calculator developed using HTML, CSS and JavaScript for the CodeAlpha Frontend Development Internship.

---

# 📌 Project Description

This is a responsive Modern Calculator developed using HTML, CSS, and JavaScript. It provides a clean and attractive user interface with glassmorphism effects and supports basic arithmetic calculations. The calculator is fully responsive and works smoothly on desktop and mobile devices.

---

# 🚀 Features

- Responsive Design
- Modern Glassmorphism UI
- Gradient Background
- Basic Arithmetic Operations
- Addition
- Subtraction
- Multiplication
- Division
- Percentage Calculation
- Decimal Support
- AC (Clear All)
- DEL (Delete Last Character)
- Error Handling for Invalid Expressions
- Smooth Hover Effects
- Mobile Friendly Layout
- Clean and Attractive User Interface

---

# 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- CSS Grid
- CSS Flexbox
- Google Fonts
- CSS Animations
- Media Queries
